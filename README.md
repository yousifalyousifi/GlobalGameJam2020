# Road Fixer 
## (Global Game Jam 2020 submission)

Typescript + Phaser 3 game.

**Theme:** Repair.

#### Deveploment environment: 
Node v12.14.1 or higher

npm v6.13.4 or higher

yarn 1.9.2 or higher


#### Set up
`yarn install`

`yarn dev` (serves files on localhost:8080)
